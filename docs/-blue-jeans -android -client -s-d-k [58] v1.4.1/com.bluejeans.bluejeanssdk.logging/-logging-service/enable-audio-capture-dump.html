enableAudioCaptureDump

fun enableAudioCaptureDump(enable: Boolean)
Content copied to clipboard

Enables Audio dump for debug purposes while in a meeting. The audio packet dump created will be placed inside the internal app directory and uploaded to BlueJeans log server and logs, on calling the uploadLog API. Once the upload is successful these will be cleared from the internal memory. Use this when you face any audio issues such as echo, no audio on send/receive, etc and keep the audio capture enabled for a minimum of 2 min and then call uploadLog API This will help debug audio-related issues on the hardware.
Note that the audio capture dumps will be heavier in size and use it only when needed, just for 2 min and ensure to upload them once captured.

Parameters

enable

true to enable and false to disable.
